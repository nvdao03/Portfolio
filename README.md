# Portfolio by Van Dao

## Requirements

- **nodejs**
- **yarn**
- **npm**

## Clone project and run test

```
$ git clone https://github.com/nvdao03/Portfolio.git
$ cd Portfolio
$ npm install
$ npm run dev
$ cd server 
$ npm run
```
